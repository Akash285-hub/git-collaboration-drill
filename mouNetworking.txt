1. netstat - Display network connection, routing tables, and interface statistics.

2. route - Displays or modifies the IP routing table.

3. ifconfig - Displays all current TCP/IP network configurations.

4. nslookup-

    nslookup [hostname] -> to obtain domain name or IP address

5. traceroute - Used to find network delays
    traceroute <destination>

6. arp - (Address resolution protocol) Used to see and include content in the ARP table
